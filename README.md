# kif-parser

WIP 作業中です。  

## Attention（注意）

初めに注意。  
棋譜ファイルはバックアップを残しておいてください。  
このアプリケーションでは、棋譜ファイルのコピーを利用してください。  

## Set up（設定）

```shell
python.exe is_ready.py
# または
is_ready.py
```

👆 [Python3](https://www.python.org/)をインストールして、`is_ready.py` スクリプトを実行してください

```shell
readyok
```

👆 問題なければ `readyok` と表示されます
